## 🎯 版本概要

### 核心特性
1. **多代理支持**: 支持 Claude Code、GitHub Codex、Gemini Code
2. **智能检测**: 基于 User-Agent 和请求头的代理类型识别
3. **透传架构**: 完整转发请求响应，不修改内容
4. **用户体验**: 现代 CLI 界面，进度条，交互式向导
5. **统一配置**: 继续使用单一 octopus.toml 配置文件

### 技术原则
- ✅ **简单透传**: 保持请求转发的简洁性
- ✅ **向后兼容**: 100% 兼容 v0.0.x 配置和命令  
- ✅ **单一配置**: 不引入多配置文件复杂性
- ✅ **TDD 开发**: 继续严格的测试驱动开发
